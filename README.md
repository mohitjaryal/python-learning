## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohitjaryal/python-learning.git
   cd python-learning
   ```

2. **Run any script**
   ```bash
   python hello.py
   python calculator.py
   ```

> Make sure you have [Python 3](https://www.python.org/downloads/) installed.
